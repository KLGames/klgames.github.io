******************************************************************
Material updater - Unity URP to HDRP README
******************************************************************

Tested in Unity 2022.3.0 on a 150 GB project, on over 7000+ materials

*********************************
HOW THIS WORK?
*********************************

0, First step is to make a backup of your URP project!

1, The next step is to copy your URP project to a new folder, lets say this name will be

YOUR_PROJECT_URP (this stay the urp project)
and you create a HDRP too:
YOUR_PROJECT_HDRP (this will be the HDRP project)

2, Open the HDRP project.

3, Install the High definition render pipeline from package manager, and fixes everything with the HDRP Wizard until everything will be GREEN. Maybe the wizard trying to update all your materials here, but you get every material as pink in the end.

4, Remove the URP renderers from your HDRP folder.

5, Copy the "Assets\KLGames\MaterialUpdaterURP" directory to your assets folder (both in URP and HDRP folder in your project)

6, Open your URP project and run the first script in the menu: "*KLGAMES/MATERIALUPDATER/-> IN URP PROJECT: SAVE MATERIAL INFO!"

This will create a json file from your material properties in the assets folder. "MaterialInfoUrp.txt" 7, After this completes, copy this file to your HDRP Asset folder ("MaterialInfoUrp.txt") //press CTRL + R for manual refresh

8, In your HDRP project click on the menu: "*KLGAMES/MATERIALUPDATER/-> IN HDRP PROJECT: UPDATE MATERIALS!"

9, After all, all your materials will be usable again!
